Okay, now that you're through the pitch, here are some interesting tidbits on the compressor that deeper users will appreciate:

  • The plugin does do some crude level matching on its own, but there are input and output knobs to dial it in further. The knobs can be linked:
    • Mac: Cmd + drag
    • Windows: Ctrl + drag
  • Button labels pop up if you hover over a button for a second.
  • Disengage all of the ratio buttons and you can drive only the tube for some saturation.
  • Engaging all of the ratio buttons enters limit mode (limit mode ignores the auto release button, fyi).
