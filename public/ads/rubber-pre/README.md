# Rubber.PRE Ad Campaign - A/B Testing

## Directory Structure

- `variant-a/` - First video variant for A/B testing
- `variant-b/` - Second video variant for A/B testing
- `variant-c/` - Third video variant for A/B testing

## File Naming Convention

Place your video files in each variant folder with descriptive names:
- `rubber-pre-ad-v1.mp4` (or .mov, .webm)
- Include platform-specific versions if needed (e.g., `instagram-9x16.mp4`, `youtube-16x9.mp4`)

## Video Specifications by Platform

### Instagram/Facebook
- **Feed**: 1080x1080px (1:1) or 1080x1350px (4:5)
- **Stories/Reels**: 1080x1920px (9:16)
- **Max duration**: 60s (Feed), 90s (Stories/Reels)
- **Format**: MP4 or MOV

### TikTok
- **Dimensions**: 1080x1920px (9:16)
- **Max duration**: 60s
- **Format**: MP4 or MOV

### YouTube
- **Dimensions**: 1920x1080px (16:9)
- **Max duration**: varies by ad type
- **Format**: MP4, MOV, or AVI

## Tracking Results

Track performance metrics for each variant:
- Click-through rate (CTR)
- Conversion rate
- Cost per acquisition (CPA)
- View completion rate
- Engagement metrics

Update the tracking spreadsheet: `ad-performance-tracking.md`
